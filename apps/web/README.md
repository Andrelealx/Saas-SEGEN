# Portal Web

Frontend inicial completo do Sistema Central da MAJ Software.

## Funcionalidades implementadas

1. Login multi-tenant.
2. Dashboard executivo.
3. CRM com funil.
4. Projetos e status.
5. Propostas comerciais.
6. Financeiro (faturas).
7. Suporte (tickets e mensagens).
8. Catalogo de produtos white-label.
9. Branding por tenant (nome e cores).

## Execucao

1. Suba a API em `apps/api`.
2. Acesse `http://localhost:3333/app`.
3. Login padrao:
   - Email: `admin@softhouse.com`
   - Senha: `admin123`
   - Tenant: `dev-tenant`
