# Modelo de Negócio

## Serviços principais

1. Sites institucionais e landing pages.
2. Sistemas web administrativos.
3. Apps e PWAs focados em operação de PME.
4. Integrações e automações.

## Estratégia de oferta

1. Produto white-label (rápido, previsível, ticket médio acessível).
2. Projeto sob encomenda (alto valor, maior complexidade).
3. Plano de suporte recorrente (estabilidade financeira).

## Pacotes comerciais sugeridos

| Pacote | Indicado para | Entrega | Prazo | Modelo de preço |
| --- | --- | --- | --- | --- |
| Start WL | PME iniciando digitalização | Sistema pronto + identidade visual | 7-15 dias | Setup + mensalidade |
| Pro WL | PME com operação ativa | WL + integrações básicas | 15-30 dias | Setup + mensalidade maior |
| Custom | Regras específicas | Projeto sob medida | 30-120 dias | Entrada + marcos |

## Precificação base (estrutura)

1. Setup: cobre descoberta, personalização, implantação e treinamento.
2. Mensalidade: cobre suporte, monitoramento, pequenas evoluções e infraestrutura.
3. Hora técnica avulsa: usada para solicitações fora de escopo.

## Critérios de escopo

1. O que entra no pacote padrão.
2. O que é customização.
3. O que é novo desenvolvimento.
4. Como aprovar mudança de escopo (change request).

## Canais de aquisição

1. Indicação de clientes.
2. Conteúdo técnico e casos no LinkedIn/Instagram.
3. Parcerias com agências de marketing e contadores.
4. Prospecção outbound segmentada por nicho.
